- Проектирование мультиагентных AI-конвейеров с контрольными точками и чекпоинтами

- Мультиагентная оркестрация: до 14 AI-агентов в одном конвейере (Claude, Gemini)

- MCP-серверы: FastMCP, 13 инструментов, граф-БД Kuzu, Streamlit-дашборды

- Промпт-инженерия: Jinja2-шаблоны, 23+ ролей агентов, мультимодельная оркестрация

### Full-Stack Development
- **Frontend:** React 19, SvelteKit 2, Astro 5, TypeScript 5.8, Tailwind CSS, Vite
- **Backend:** Firebase (Auth, Firestore, Cloud Functions), FastAPI, Python (httpx, Pydantic)
- **Базы данных:** Kuzu (графовая), Firestore (NoSQL), MySQL 8, Chroma (векторная)
- **DevOps:** Docker Compose, GitHub Actions, Nginx reverse proxy, CI/CD-конвейеры
- **API:** Google Gemini, Tavily, Serper, Google Knowledge Graph, Wikidata SPARQL

### SEO и контент-автоматизация
- Entity-based SEO (анализ Knowledge Graph, классификация сущностей)
- Автоматизированное SEO-исследование (конкуренты, кластеризация ключевиков, LSI)
- Конвейеры фактчекинга с верификацией источников
- Платформенная адаптация контента (Яндекс.Дзен, региональный таргетинг)

## Проекты

### MediaUniversal — Мультиагентный AI-конвейер контента
**AI Pipeline | Python, Claude CLI, Tavily, Pydantic, pytest | Sales-Ready: 8/10**

Флагманский проект. Полностью автоматизирует создание SEO-оптимизированных статей: от анализа конкурентов до публикации. 14+ AI-агентов, отказоустойчивая архитектура, 1 592 строки тестов.

- Заменяет команду из 4-5 человек (SEO-аналитик, копирайтер, фактчекер, редактор, мета-специалист)
- Сокращение времени: 14-20 часов вручную → ~30 минут (~96%)
- 6 автоматизированных этапов: Исследование → Проектирование → Черновик → Фактаудит → Редактура (6 агентов) → SEO-мета

### Entity SEO Intelligence (ESI) — Knowledge Graph MCP Server
**MCP Server | Python, FastMCP, Kuzu, Streamlit, Google KG API | Sales-Ready: 7/10**

13 MCP-инструментов для entity-based SEO. Анализирует, как Google воспринимает бренд через сущности Knowledge Graph. Граф-БД Kuzu, 115 тестов, дашборд на Streamlit.

- Экономия ~20-30 часов аналитической работы на один бренд-аудит
- Мониторинг бренда: автоматические снапшоты, diff с предыдущим состоянием, пороговые оповещения
- Планирование контента на основе классификации сущностей (pillar/cluster/supporting)

### Pereplan (GSD) — SEO-ориентированный корпоративный сайт
**Static Website | Astro 5, TypeScript, Tailwind CSS, GitHub Actions | Sales-Ready: 9/10**

Production-сайт компании «АПМ-1» на soglasovanie.su. 50+ страниц, 34 услуги, полный CI/CD-конвейер: push → build → FTP → smoke test. Основной канал привлечения клиентов через органический поиск.

- CI/CD: GitHub Actions → FTP Deploy → Smoke Test (5 страниц)
- SEO: Sitemap, RSS, JSON-LD (Organization, BlogPosting, LocalBusiness), OG-теги
- Лидогенерация: Formspree + honeypot-защита + маска телефона

### UXKit (Яндекс.Аренда) — Production-дизайн-система
**Design System | Tailwind CSS, PostCSS, Atomic Design | Sales-Ready: 8/10**

Полная дизайн-система для платформы Яндекс.Аренда: 30+ компонентов (12 атомов, 12 молекул, 8+ организмов), WCAG 2.1 AA, 10 227 строк документации.

- Ускоряет разработку UI в 3-5 раз — экономия 10-15 ч/неделю
- Accessibility: контраст ~17.8:1, клавиатурная навигация, ARIA, тач 44x44px

### Reviews-WB — Мультиагентная обработка отзывов маркетплейсов
**AI Pipeline | Python, FastAPI, Claude SDK, SQLite | Sales-Ready: 7/10**

6 AI-агентов (Claude Haiku + Sonnet) автоматизируют ответы на отзывы Wildberries/Ozon. Sentiment routing, валидация галлюцинаций, веб-интерфейс. В 10-20 раз быстрее ручной работы.

- ~97% экономия: 30-60 сек vs 8-12 мин вручную на каждый отзыв
- Негативные отзывы → одобрение менеджера; позитивные → авто-публикация

### Gemini Light RAG 2.0 — Командная база знаний с AI-чатом
**Full-Stack App | React 19, TypeScript, Firebase, Google Gemini API | Sales-Ready: 6/10**

Команда (2-10 человек) загружает документы и задаёт вопросы через AI-чат. Ролевой доступ, Cloud Functions, экспорт в Markdown.

- Экономия 2-4 часа/день на поиск в документации для команды из 5 человек
- AI-ответы с цитированием источников, автоматическая генерация заголовков чатов

## Методология работы

- **Solo + AI-ассистенты:** все проекты реализованы самостоятельно с Claude Code и Cursor
- **Итеративная разработка:** фазовый подход (MediaUniversal: N8N → Python, ESI: 8 фаз, RAG 2.0: 6 фаз)
- **Качество через тесты:** pytest, структурированное логирование, отказоустойчивое управление состоянием
- **Documentation-First:** CLAUDE.md, PRD.md, QUICKSTART.md, GUIDE.md для каждого проекта
- **CI/CD:** GitHub Actions, автоматические деплои, линтинг и проверки на каждый коммит
- **Design Systems:** Tailwind + токены, единая типографика, адаптивные сетки, тёмная/светлая тема

## Экосистема проектов

Проекты не изолированы — они образуют связанную систему:

- **ESI** → **Copiwriter** / **MediaUniversal** — данные сущностей и SEO-инсайты
- **Copiwriter** → **MediaUniversal** — эволюция: Claude Skills → Python-конвейер
- **Veritas** → **Copiwriter** / **MediaUniversal** — верификация фактов
- **Embedding Google** → **Gemini RAG 2.0** — эволюция: v1 → v2 (команда + Cloud Functions)
- **TextRu** → **Dzen** → **Pereplan** — проверка плагиата → адаптация контента → сайт
- **UXKit** → **Site3-0** — дизайн-система → фронтенд headless CMS
